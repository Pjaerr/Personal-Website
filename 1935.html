Project 1935 (On Hold)

Go Back

Project 1935 is a grand strategy multiplayer game featuring up to 12 players that uses UNET in a real-time fashion where the game can be left and returned to, with all actions still ongoing. Designed to be played like a digital board game, similar to RISK or Diplomacy.

I have included this on my portfolio as a lot of work has been completed on it, however it is currently on hold in favour of completing many smaller projects first. Some of the devlogs for this project can be found at the Gamejolt link.

Made with Unity and C#.

Timespan: On Hold.
